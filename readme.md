# Task Flow

Task Flow is a Spring Boot backend application for managing user tasks with role-based access control, JWT authentication, and scalable REST APIs.

## Features
- JWT Authentication & Authorization  
- Role-Based Access Control (Admin/User)  
- Admin can assign tasks to any user  
- Users can manage only their own tasks  
- Pagination & sorting for all list APIs  
- Global exception handling  

## Tech Stack
- Java, Spring Boot, Spring Security  
- MySQL  
- JWT, Maven
